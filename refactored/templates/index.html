EMODnet Seabed Habitats - EUSeaMap Layers

API Endpoints

Layer Information

GET /api/layers ✓ Working
Returns all available EUSeaMap layers (45 layers)
GET /api/all-layers ✓ Working
Returns comprehensive layer information including WMS and HELCOM
GET /api/capabilities ✓ Working
Returns WMS GetCapabilities document

Current Configuration

  • Data Source: EMODnet Seabed Habitats
  • Filter: EUSeaMap layers only (layers starting with 'eusm')
  • Total Layers: 45 EUSeaMap layers from 271 total available
  • Layer Types: Biological zones, Substrate types, Energy classes, EUNIS classifications
  • Years: EUSeaMap 2019, 2023

Layer Operations

GET /api/legend/<layer_name>
Get legend URL for a specific layer
POST /api/feature-info
Get feature information for a specific location

Example Usage

# Get all EUSeaMap layers
curl http://localhost:5000/api/layers

# Get comprehensive layer information
curl http://localhost:5000/api/all-layers

# Get WMS capabilities
curl http://localhost:5000/api/capabilities
